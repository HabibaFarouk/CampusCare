# GIU Campus Maintenance Mobile App

A mobile application for reporting and managing campus maintenance issues at the German International University (GIU).

## Features
- Community members can submit, view, edit, and track maintenance tickets.
- Facility Management staff can assign, update, and close tickets.
- Workers can view assigned tasks, update status, and upload completion photos.
- Role-based access control ensures security.
- Push notifications for real-time updates.

## Technology Stack
- **Frontend:** React Native (Expo)
- **Backend:** Node.js with RESTful APIs
- **Database:** PostgreSQL, managed via Supabase
- **Cloud:** Cloud storage for images, secure hosting, and backups
